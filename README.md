### Prerequisites

- Python 2.7 or 3.6
- Pytorch 1.1 or higher
- CUDA 9.0 or higher
- tensorboardX

### Preparation

First of all, clone the code
```
git clone https://github.com/leowangzi/CascadeRCNN.git
cd CascadeRCNN
```
Then, create a folder:
```
mkdir data
```
or
```
ln -s [source_data] data
```
